# myApp

Plantilla base en Node.js para construir una web app sencilla sin Wasp.

## Correr en local

```bash
cd /Users/deyli/myApp
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Estructura

- `server.js`: servidor Node y rutas API basicas
- `public/index.html`: landing inicial
- `public/styles.css`: estilos
- `public/app.js`: ejemplo de llamada al backend
